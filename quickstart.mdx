# Quick Start

This guide will walk you through creating your first haptic feedback interaction with WebHaptics.

## Basic Usage

<Steps>
  <Step title="Import the hook or function">
    Import the appropriate hook for your framework:

    <CodeGroup>
    ```tsx React
    import { useWebHaptics } from 'web-haptics/react';
    ```

    ```ts Vue
    import { useWebHaptics } from 'web-haptics/vue';
    ```

    ```ts Svelte
    import { createWebHaptics } from 'web-haptics/svelte';
    ```

    ```ts Vanilla
    import { WebHaptics } from 'web-haptics';
    ```
    </CodeGroup>
  </Step>

  <Step title="Initialize WebHaptics">
    Create an instance or use the hook in your component:

    <CodeGroup>
    ```tsx React
    function MyComponent() {
      const { trigger, isSupported } = useWebHaptics();
      // ...
    }
    ```

    ```vue Vue
    <script setup>
    const { trigger, isSupported } = useWebHaptics();
    </script>
    ```

    ```svelte Svelte
    <script>
      const { trigger, isSupported } = createWebHaptics();
    </script>
    ```

    ```ts Vanilla
    const haptics = new WebHaptics();
    const isSupported = WebHaptics.isSupported;
    ```
    </CodeGroup>
  </Step>

  <Step title="Trigger haptic feedback">
    Call `trigger()` with a preset name or custom pattern:

    <CodeGroup>
    ```tsx React
    <button onClick={() => trigger('success')}>
      Submit
    </button>
    ```

    ```vue Vue
    <button @click="trigger('success')">
      Submit
    </button>
    ```

    ```svelte Svelte
    <button on:click={() => trigger('success')}>
      Submit
    </button>
    ```

    ```ts Vanilla
    button.addEventListener('click', () => {
      haptics.trigger('success');
    });
    ```
    </CodeGroup>
  </Step>
</Steps>

## Using Built-in Presets

WebHaptics includes 11 pre-configured haptic patterns. Here's how to use them:

<CodeGroup>
```tsx React
import { useWebHaptics } from 'web-haptics/react';

function NotificationExample() {
  const { trigger } = useWebHaptics();

  return (
    <div>
      <button onClick={() => trigger('success')}>
        Success
      </button>
      <button onClick={() => trigger('warning')}>
        Warning
      </button>
      <button onClick={() => trigger('error')}>
        Error
      </button>
    </div>
  );
}
```

```vue Vue
<script setup>
import { useWebHaptics } from 'web-haptics/vue';

const { trigger } = useWebHaptics();
</script>

<template>
  <div>
    <button @click="trigger('success')">
      Success
    </button>
    <button @click="trigger('warning')">
      Warning
    </button>
    <button @click="trigger('error')">
      Error
    </button>
  </div>
</template>
```

```svelte Svelte
<script>
  import { createWebHaptics } from 'web-haptics/svelte';
  
  const { trigger } = createWebHaptics();
</script>

<div>
  <button on:click={() => trigger('success')}>
    Success
  </button>
  <button on:click={() => trigger('warning')}>
    Warning
  </button>
  <button on:click={() => trigger('error')}>
    Error
  </button>
</div>
```
</CodeGroup>

### Available Presets

| Preset | Description | Use Case |
|--------|-------------|----------|
| `success` | Ascending double-tap | Successful operations |
| `warning` | Two taps with hesitation | Warning messages |
| `error` | Three rapid harsh taps | Error states |
| `light` | Single light tap | Minor interactions |
| `medium` | Moderate tap | Standard interactions |
| `heavy` | Strong tap | Significant actions |
| `soft` | Soft, cushioned tap | Gentle feedback |
| `rigid` | Hard, crisp tap | Precise actions |
| `selection` | Subtle tap | Selection changes |
| `nudge` | Two quick taps with pause | Reminders |
| `buzz` | Continuous vibration | Alerts |

## Custom Patterns

Create your own haptic patterns for unique interactions:

<CodeGroup>
```tsx React
import { useWebHaptics } from 'web-haptics/react';

function CustomPattern() {
  const { trigger } = useWebHaptics();

  const customPattern = [
    { duration: 30, intensity: 0.5 },
    { delay: 60, duration: 40, intensity: 1 },
    { delay: 80, duration: 20, intensity: 0.3 }
  ];

  return (
    <button onClick={() => trigger(customPattern)}>
      Custom Feedback
    </button>
  );
}
```

```vue Vue
<script setup>
import { useWebHaptics } from 'web-haptics/vue';

const { trigger } = useWebHaptics();

const customPattern = [
  { duration: 30, intensity: 0.5 },
  { delay: 60, duration: 40, intensity: 1 },
  { delay: 80, duration: 20, intensity: 0.3 }
];
</script>

<template>
  <button @click="trigger(customPattern)">
    Custom Feedback
  </button>
</template>
```

```svelte Svelte
<script>
  import { createWebHaptics } from 'web-haptics/svelte';
  
  const { trigger } = createWebHaptics();
  
  const customPattern = [
    { duration: 30, intensity: 0.5 },
    { delay: 60, duration: 40, intensity: 1 },
    { delay: 80, duration: 20, intensity: 0.3 }
  ];
</script>

<button on:click={() => trigger(customPattern)}>
  Custom Feedback
</button>
```
</CodeGroup>

<Note>
  Pattern durations are in milliseconds. Intensity ranges from 0 (off) to 1 (maximum).
</Note>

## Adjusting Intensity

Control the overall intensity of any haptic pattern:

<CodeGroup>
```tsx React
// Subtle success feedback
trigger('success', { intensity: 0.3 });

// Strong error feedback
trigger('error', { intensity: 1 });
```

```vue Vue
// Subtle success feedback
trigger('success', { intensity: 0.3 });

// Strong error feedback
trigger('error', { intensity: 1 });
```

```svelte Svelte
// Subtle success feedback
trigger('success', { intensity: 0.3 });

// Strong error feedback
trigger('error', { intensity: 1 });
```
</CodeGroup>

## Checking Support

Always check if haptics are supported before showing UI that depends on it:

<CodeGroup>
```tsx React
import { useWebHaptics } from 'web-haptics/react';

function App() {
  const { trigger, isSupported } = useWebHaptics();

  return (
    <div>
      {isSupported ? (
        <button onClick={() => trigger('medium')}>
          Tap for feedback
        </button>
      ) : (
        <p>Haptics not available on this device</p>
      )}
    </div>
  );
}
```

```vue Vue
<script setup>
import { useWebHaptics } from 'web-haptics/vue';

const { trigger, isSupported } = useWebHaptics();
</script>

<template>
  <div>
    <button v-if="isSupported" @click="trigger('medium')">
      Tap for feedback
    </button>
    <p v-else>
      Haptics not available on this device
    </p>
  </div>
</template>
```

```svelte Svelte
<script>
  import { createWebHaptics } from 'web-haptics/svelte';
  
  const { trigger, isSupported } = createWebHaptics();
</script>

<div>
  {#if isSupported}
    <button on:click={() => trigger('medium')}>
      Tap for feedback
    </button>
  {:else}
    <p>Haptics not available on this device</p>
  {/if}
</div>
```
</CodeGroup>

<CodeGroup>




</CodeGroup>

<Tip>
  Debug mode adds audio feedback that simulates the haptic patterns, making it easier to develop and test on desktop.

## Complete Example

Here's a full form example with haptic feedback:

<CodeGroup>
```tsx React
import { useWebHaptics } from 'web-haptics/react';
import { useState } from 'react';

function LoginForm() {
  const { trigger } = useWebHaptics();
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');

  const handleSubmit = async (e) => {
    e.preventDefault();
    trigger('light'); // Immediate feedback

    try {
      await login(email, password);
      trigger('success');
    } catch (error) {
      trigger('error');
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        onFocus={() => trigger('selection')}
      />
      <input
        type="password"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
        onFocus={() => trigger('selection')}
      />
      <button type="submit">Login</button>
    </form>
  );
}
```

```vue Vue
<script setup>
import { useWebHaptics } from 'web-haptics/vue';
import { ref } from 'vue';

const { trigger } = useWebHaptics();
const email = ref('');
const password = ref('');

const handleSubmit = async () => {
  trigger('light'); // Immediate feedback

  try {
    await login(email.value, password.value);
    trigger('success');
  } catch (error) {
    trigger('error');
  }
};
</script>

<template>
  <form @submit.prevent="handleSubmit">
    <input
      v-model="email"
      type="email"
      @focus="trigger('selection')"
    />
    <input
      v-model="password"
      type="password"
      @focus="trigger('selection')"
    />
    <button type="submit">Login</button>
  </form>
</template>
```

```svelte Svelte
<script>
  import { createWebHaptics } from 'web-haptics/svelte';
  
  const { trigger } = createWebHaptics();
  let email = '';
  let password = '';
  
  async function handleSubmit(e) {
    e.preventDefault();
    trigger('light'); // Immediate feedback
    
    try {
      await login(email, password);
      trigger('success');
    } catch (error) {
      trigger('error');
    }
  }
</script>

<form on:submit={handleSubmit}>
  <input
    bind:value={email}
    type="email"
    on:focus={() => trigger('selection')}
  />
  <input
    bind:value={password}
    type="password"
    on:focus={() => trigger('selection')}
  />
  <button type="submit">Login</button>
</form>
```
</CodeGroup>
